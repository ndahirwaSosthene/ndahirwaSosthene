## In summary...

I build software around a simple philosophy: developer experience (DX) and user experience (UX) are inseparable. A good system should be invisible—solving the problem entirely while using only the resources it absolutely needs. Whether parsing offline transaction data or structuring relational databases, I aim for zero bloat, clean interfaces, and self-contained logic that doesn't leak complexity.

## Featured Projects

### Leji • [[repo](https://github.com/ndahirwaSosthene/leji)]
I built this to auto-capture mobile money transactions directly from SMS notifications, removing the need for manual tracking via local background parsing.
* **Tech:** Flutter, Dart, Local Persistence

### MediflowDB • [[repo](https://github.com/ndahirwaSosthene/MediflowDB)]
I designed a 16-table relational schema representing a complete healthcare ecosystem with custom T-SQL stored procedures. When many-to-many relations got complex mid-build, I pivoted and added custom bridge tables to keep queries optimized under a strict deadline.
* **Tech:** SQL Server, T-SQL, RBAC

### Mediflow • [[live](https://mediflow-five.vercel.app/landing)]
We built this platform to connect patients, pharmacies, and insurers under a single unified API to solve slow medication access. Built in a 48-hour hackathon with a team of five, it took **5th Place Global** at the Global Entrepreneurship Week Hackathon.
* **Tech:** Flutter, API Integration

### Research Data Pipeline [[repo](https://github.com/ndahirwaSosthene/arj-validation)]
I built an automated validation and outreach pipeline that processes 256 contacts per run and pipes real-time analytics to UNDP stakeholders, replacing manual data logging entirely.
* **Tech:** Google Apps Script, Zapier, Webhooks
